# Listen & Tap the ABC

A child-friendly alphabet listening game with:

- Sequential A–Z mode
- Random mixed-letter mode
- Easy animal identification game
- 8 familiar animals with pictures and genuine animal recordings
- Feminine text-to-speech voice
- Musical letter prompts
- Pictures and spoken words after correct answers
- Offline and Home Screen support

Open `index.html` in a browser, or serve the directory over HTTPS to install it as a web app.
